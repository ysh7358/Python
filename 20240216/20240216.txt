---------------------------------------------
논리연산자

and - 두 개가 모두 참이어야 참 / 나머지는 거짓
or - 둘 중 하나라도 참이면 참 / 두 개 모두가 거짓일 경우 거짓
not - 반대의 값
----------------------------------------------

[and] 논리곱 (0은 거짓, 1은 참)
x y z(결과)	x	y	결과
0 0 0		거짓	거짓	거짓
0 1 0		거짓	참	거짓
1 0 0		참	거짓	거짓
1 1 1		참	참	참

[or] 논리합 (0은 거짓, 1은 참)
x y z(결과)	x	y	결과
0 0 0		거짓	거짓	거짓
0 1 1		거짓	참	참
1 0 1		참	거짓	참
1 1 1		참	참	참

-------------------------------------------

조건문 if / elif / else
if - 만약
elif - 만약 블록을 벗어나서 여기에 해당되면 여기
else - 싹 다 아니면 여기
