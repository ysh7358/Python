-------------------------------------------

리스트에 요소 추가
1) append - 맨 끝자리에 데이터가 들어감
리스트명.append(항목)
ex) list.append('hi')

2) insert - 데이터를 추가할 위치
리스트명.insert(삽입할 위치, '항목')
ex) list.insert(2, 'hi')

--------------------------------------------

리스트명[a : b] : a부터 b-1까지 슬라이싱
a[x : y] : x부터 y-1까지
a[0 : 3] : 0부터 2까지

--------------------------------------------

요소 확인(in)
a = "청년취업사관학교"
"청년" in a > true
"치킨" in a > false

리스트 요소 확인

-------------------------------------------

리스트의 요소 변경

cart =['사과', '세제', '화장지', '치약']
cart[1] = ['섬유 유연제']
print(cart)
['사과', '섬유 유연제', '화장지', '치약']
리스트의 인덱스 범위 안에서 변경 가능
범위 밖에서 시도할 경우 out of range 오류
